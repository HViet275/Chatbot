OS: Windows 10/11
Python: 3.11.6 

MỞ Command Promt TẠI THƯ MỤC NÀY VÀ THỰC HIỆN CÁC BƯỚC SAU

BƯỚC 1: Tải các thư viện để chạy file
-- Tại vị trí thư mục source gõ câu lệnh sau vào Command Promt
pip install -r requirements.txt

BƯỚC 2: Dùng streamlit để chạy file Data.py trong terminal Python
streamlit run Data.py

BƯỚC 3: Chỗ Data sẽ hiện thị tập dữ liệu, có thể xem các thông số ứng dụng đã được làm rõ bên trên như Data Processing, Convert Data, Optimizer Model, Train – Test Model, ….

BƯỚC 4: Khi nhấn vào ChatBot, chọn lĩnh vực muốn đặt câu hỏi. Nhập câu hỏi vào textbox, Sau đó nhấn Send. Chatbot sẽ hiển thị câu trả lời của lĩnh vực đó, Nếu câu hỏi đặt ra không nằm trong lĩnh vực thì Chatbot trả lời "Tôi chưa học về nó, sẽ cập nhật sau."

BƯỚC 5: Nếu muốn hỏi câu hỏi khác, nhấn Clear để xóa câu trả lời và đặt lại câu hỏi mới.
